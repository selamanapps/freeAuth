Rebrand Your Telegram Bot

Configure your bot's visual identity through BotFather. A professional appearance builds user trust and credibility, especially when asking for phone verification.

Prerequisites

  1. 1 Open Telegram app
  2. 2 Search for @BotFather (blue checkmark)
  3. 3 Type /mybots and select your bot
/setname

Bot Name

The title displayed in chat list and header. Should be clear and professional.

Recommended: AppName | Action

Example: FreeAuth | Secure Verification

/setdescription

Description

The landing page users see before clicking "Start". Explain why and that it's safe.

/setabouttext

About Text

Text shown when user clicks on bot profile. Provides credibility and support info.

Official automated verification gateway for your app. Powered by FreeAuth & Telegram.

Support: @your_support_username

/setuserpic

Profile Picture

The icon in the chat list. A default letter icon looks unprofessional.

Specs: Minimum 512x512 pixels

Design tip: Use bold icon on solid background. Avoid small text.

Idea: White shield with checkmark on blue background

Command List

Optional menu button next to text input for better UX.

start - Begin verification help - Get support privacy - Read privacy policy

Pro Tip: File IDs for Code

Use specific images in your server without external hosting

  1. 1. Send image to bot in private chat
  2. 2. Forward message to @JsonDumpBot
  3. 3. Copy file_id from JSON
  4. 4. Use string in ASSETS object instead of URL
